# Orbit Agent — Tester Dashboard

**Live:** https://nekrut.github.io/orbit-dashboard/

A public, **anonymized** dashboard for the [Orbit Agent](https://galaxyproject.org/tools/orbit/)
beta — the Galaxy Project's AI co-scientist for bioinformatics. It shows each tester's activity
(bug reports filed through Orbit's in-app Feedback button) and the `galaxyproject/loom` issues and
pull requests their feedback turned into.

Testers appear by an anonymous ID (e.g. `orbit-007`) only — no names or emails. The page is
generated automatically from the beta tester program and refreshes a few times a day.
